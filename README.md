## 🎮 功能特点

- ✅ 完整的中国象棋规则实现
- ✅ 人机对战模式
- ✅ 双人对战模式
- ✅ 棋局保存与加载
- ✅ 悔棋功能
- ✅ 多种难度级别

---

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/520llw/-.git
cd -

# 安装依赖
pip install -r requirements.txt

# 启动游戏
python main.py
```

---

## 🎯 游戏模式

1. **人机对战** - 与AI对弈
2. **双人对战** - 本地双人对弈
3. **残局挑战** - 经典残局破解

---

## 📝 技术栈

- Python 3.8+
- Pygame（图形界面）
- Minimax算法（AI）
